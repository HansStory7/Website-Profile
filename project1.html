Project Overview

Thé Ciliwung Tea Estate is a website project aimed at introducing and promoting the tourist destinations along the Ciliwung River. This website is designed to be a digital guide for tourists who want to explore the natural beauty and culture around this historic river.

Built with basic web technologies like HTML, CSS, and JavaScript, this project focuses on clear information delivery, easy navigation, and responsive design. You can visit the site directly at https://hansstory7.github.io/The-Ciliwung/.

Key Features

Interactive Map

Displays important tourist locations along the river for easy exploration.

Destination Gallery

A collection of attractive photos and descriptions for each featured tourist spot.

Responsive Design

The site's layout adapts well to various devices, from desktops to mobile phones.

Practical Information

Provides essential information such as opening hours, ticket prices, and visiting tips for tourists.
