Project Overview

This project is a personal portfolio website for Farhan Abdullah, designed to serve as a digital CV and showcase his professional profile. Built with Canva, the site features a clean, minimalist design that puts the focus on content and readability. It effectively communicates his skills, educational background, and contact information in a single, accessible page.

The primary goal was to create a professional online presence that is both informative and visually appealing, allowing potential employers or collaborators to quickly get a comprehensive overview of his capabilities.

Key Features

Professional Branding

A consistent and clean design that reflects a professional image and personal brand.

Single-Page Layout

All essential information is presented in a streamlined, easy-to-navigate single-page format.

Integrated Social Links

Direct links to professional profiles and social media for easy networking and contact.

Built with Canva

Showcases the ability to create polished and functional websites using modern design tools like Canva.
